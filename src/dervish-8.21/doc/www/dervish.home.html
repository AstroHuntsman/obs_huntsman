Dervish Home Page


Dervish (formerly - Survey Human Interface and Visualization Environment) is the supported toolkit to be used for assembling many of the data reduction pipelines and the host software for the SDSS data acquisition systems.

Broadly speaking, the software serves three classes of programmers: Those wanting to use a TCL shell, those seeking to add new TCL extensions to Dervish, and programmers using other tool-kits wanting to interface with the Dervish system.

DISCLAIMER: The software referenced on these pages was developed at the Fermi National Accelerator Laboratory for use in the offline analysis of astronomical data.
