Useful Utility TCL Commands in Dervish

Dervish contains many Tcl extensions that can be used to aid a Dervish user who is writing their own Tcl procedures or trying to debug some of their own Tcl extensions. These Tcl extensions are listed below with a brief summary. For more information see the complete documentation.

  • exprPrint - handle expression printer
  • exprGet - handle expression evaluation
  • vectorExprPrint - arithmetic expression printer
  • getHistoryName - return the current value of DERVISH_HISTFILE or /dev/null.
  • handleList - list all handles, types and addresses
  • handleListFromType - list all handles of a specific type
  • handleGet - return address and type of a handle
  • handlePtr - return address of a handle
  • handleSet - handle expression set command
  • handleSetFromHandle - set a handle using another handle
  • handleType - return type of a handle
  • historyRestore - read in the history file and add it to the history list.
  • historySave - save the current history list to a file.
  • nameGetFromType - return name associated with a type
  • regListNames - list all handles which are of type REGION
  • schemaGet - list the schema associated with a handle
  • schemaGetFromType - list the schema assocated with a type
  • schemaMemberGet - return an element of a schema
  • schemaPrint - print the schema associated with a handle
  • schemaPrintFromType - print the schema associated with a type
  • timerLap - return elapsed/CPU times since timerStart
  • timerStart - start timer
  • treeGet - list all addresses that were shMalloced
  • treePrint - print all addresses that were shMalloced
  • typeGetFromName - return type associated with ascii name
  • verboseSet - set the verbosity level for debugging
