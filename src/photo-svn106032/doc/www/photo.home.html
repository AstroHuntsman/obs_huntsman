Photo Home Page

Photo (The SDSS Photometric Pipeline) is built on top of the Dervish product. It contains code that will process data from the photometric chips of the 2.5 meter telescope, and produce object lists with the parameters necessary to select targets.

The pipeline has been subdivided into two smaller pipelines: the Postage Stamp Pipeline and the Frames Pipeline. The Postage Stamp Pipeline uses small parts of the original images (which were recorded separately in the data acquisition system) to calculate all quanities that require information from more than one frame. The Frames Pipeline processes the whole images, one frame (in five colors) at a time.

The product includes C code which manipulates data and data products, and the TCL scripts that bind them together into a pipeline.
