nextNewMoonLocal


DESCRIPTION:

	The procedure returns a keyed list with the time 
        for the next new moon following an arbitraty starting
        an arbitrary longitude.  If no
        longitude is specified it defaults to the APO
        3.5 m.  The returned time is accurate to within
        a few minutes.


TCL SYNTAX:
     nextNewMoonLocal <timestamp> -longitude -help




TCL ARGUMENTS:
     <timestamp>     : starting time
     -obsLongitude   : longitude of observatory
                             default is APO 3.5 m




RETURN VALUES:

	Returns a keyed list with the date of the next new
        moon as an MJD, and MJD timestamp and an LST.
