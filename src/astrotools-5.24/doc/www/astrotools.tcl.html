ASTROTOOLS TCL Interface

List of all TCL routines in ASTROTOOLS


atExternalSet.c Preset constants


tclAirmass.c calculate airmass
tclAperture.c measure counts inside apertures
tclCali.c apply photometric calibration
tclEphemeris.c calculate rise/set times, lunar phases, hour angle
tclGalaxy.c calculate star counts
tclHgMath.c operate on HGs and HG2s (histograms, 2-D histograms)
tclImSim.c simulate images
tclLinearFits.c linear fitting
tclMatch.c match two lists of objects
atNextNewMoonLocal.tcl determine next new moon
tclObject.c measure photometric parameters
atPhotometry.tcl photometric transformations
tclPMatch.c match two CHAINs of objects, with examples
atRegContourPlot.tcl make contour plots
tclRegSci.c calculate statistics and other operations on images
tclSlalib.c use SLALIB astrometric routines
tclSurveyGeometry.c calculate coordinate transformations
atSurveyGeom.tcl calculate survey coordinates, MT patch name, stripe number
tclTrans.c create and manipulate TRANS structs
tclDustGetval.c Read values from the dust map.
