ASTROTOOLS C Interface

List of all C functions in ASTROTOOLS


atExternalSet.c preset constants


atAirmass.c calculate airmass
atAperture.c measure counts inside apertures
atCali.c apply photometric calibration
atCentroid.c determine centroids and fit Gaussians (DA)
atEphemeris.c calculate rise/set times, lunar phases, hour angle
atGalaxy.c calculate star counts
atGauss.c convolve with Gaussian
atGaussianWidth.c measure optimum sigma for smoothing, fit Gaussians (DA)
atHgMath.c operate on HGs and HG2s (histograms, 2-D histograms)
atImSim.c simulate images
atLinearFits.c linear fitting
atMatch.c match two lists of objects
atObject.c measure photometric parameters
atPMatch.c match two CHAINs of objects
atQuasar select quasar targets
atRegSci.c calculate statistics and other operations on images
atSlalib.c use SLALIB astrometric routines
atSlalib.h time stamp struct
atSurveyGeometry.c calculate survey coordinates, MT patch name, stripe number
atTrans.c create and manipulate TRANS structs
atDustGetval.c Read values from the dust map.
